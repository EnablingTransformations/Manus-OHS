# Email Retargeting Sequences for Optimal Health Summit

## Overview

Email retargeting converts 3-5x more leads than ads alone. This guide provides automated email sequences to nurture leads and drive ticket sales.

## Email Automation Setup in Mailchimp

### Step 1: Create Audience
1. Go to Audience → All Contacts
2. Create new audience "OHS Summit Leads"
3. Add form to website

### Step 2: Create Automation
1. Go to Automations → Automation
2. Select "Welcome Series"
3. Set trigger: "When someone subscribes"
4. Add emails from Sequence 1
5. Set delays between emails

### Step 3: Segment by Behavior
1. Create segment: "Purchased Ticket"
   - Condition: Clicked link to checkout
   - Condition: Visited thank-you page
   
2. Create segment: "Cart Abandoner"
   - Condition: Clicked checkout link
   - Condition: Did NOT visit thank-you page

3. Create segment: "Non-Purchaser"
   - Condition: Did NOT click checkout link

### Step 4: Assign Sequences
- Welcome Series → All new subscribers
- Cart Abandoner Series → Cart abandoners
- Post-Purchase Series → Ticket buyers
- Non-Purchaser Series → Non-buyers (5 days before event)

## Email Performance Metrics

### Key Metrics to Track
| Metric | Target | Action |
|--------|--------|--------|
| Open Rate | > 25% | If < 25%, improve subject line |
| Click Rate | > 5% | If < 5%, improve CTA or offer |
| Conversion Rate | > 2% | If < 2%, improve messaging |
| Unsubscribe Rate | < 0.5% | If > 0.5%, reduce email frequency |

### Daily Reporting
- Emails sent and delivered
- Open rate and click rate
- Conversions and revenue
- Unsubscribes and complaints
